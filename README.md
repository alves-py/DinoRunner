# DinoRunner

Só para descrever




## Installation

Install my-project with pip

```bash
  git clone https://github.com/alves-py/DinoRunner.git
  cd DinoRunner
  python -m pip install -U pygame
```
## Run my project

```bash
    start main.py
```
